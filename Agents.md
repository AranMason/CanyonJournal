CanyonJournal is a web app for canyoneers to log and track their canyon descents. See `/bots/APP_OVERVIEW.md` for full architecture, API, and database documentation.

## Directories

- `/public` — Static marketing website (homepage)
- `/routes` — Express API route handlers; the main application back-end
- `/src` — React front-end application
- `/bots` — Documentation files for AI agents to consume
- `/sql` — Scripts to create or migrate the SQL Server database


## Behaviour

Always proactively ask clarifying questions
The needs of Canyoneers will likely not align with your understanding of the sport - Be careful with making assumptions.
