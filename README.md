## Tools Used Currently

- Hosting: Heroku
- Auth: Auth0
- Front-end: React
- Back-end: Express

## Running Locally
 
 1. Copy `.env` and populate Auth0 and SQL Server credentials
 2. Install dependencies: `npm install`
 3. Start the front-end (React dev server): `npm run dev`
 4. Start the back-end (Express): `npm run watch:server`
 
 The React dev server proxies API requests to the Express server (default port `8000`).
 
 To build for production: `npm run build` then `npm start
